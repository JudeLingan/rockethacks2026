## Goals
We aim to make a robot that moves and scans its environement, returning it to the user over the internet.
The main points we aim to achieve are as follows:
- [ ] Robot that moves
- [ ] Front-End Web-App Interface
- [ ] Cyber-Security Shit
- [ ] Lidar/IR Scanner
- [ ] Make Robot Autonomous
